## English

**LS File Manager (雷神文件管理)** is a lightweight, responsive, and smart Windows desktop utility designed to optimize folder and file management. Sliding out seamlessly when your cursor hovers over the designated screen edge, LS File Manager gives you instant access to your most frequently used folders, active project files, and recently opened drawings—without cluttering your workspace.

### Key Features
- **🚀 Hover-Activated Access**: Moves instantly to the active monitor containing your cursor and slides out on edge-hover. Collapses automatically when your cursor leaves.
- **💾 Active Tab State Memory**: Automatically remembers your last active tab (Folders, Files, Recent, Custom) and restores it upon next startup.
- **📂 Split File & Directory Sorting**: Independent frequency and modification time tracking for files and directories (no scanning lags).
- **📌 Custom Pinning Shelf**: Quickly drag, add, or toggle pinned folders/files in the custom shelf for persistent single-click access. Perfect for CAD drawing workflows.
- **🌓 Adaptive Color Themes**: Supports Dark Mode, Light Mode, and Follow System Theme with real-time transition updates.
- **🌐 Bilingual UI**: Seamless English and Simplified Chinese interface switching.
- **⚡ High-Performance & Compact**: Native C# WPF implementation. Publishes into a single optimized executable under 70MB with near-zero idle CPU usage.

### Installation & Quick Start
1. Download the latest release (`SidebarApp.exe`) from the GitHub Releases page.
2. Put the executable in any directory and double-click to run.
3. Hover your mouse to the designated edge (default is **Right**) to slide out the sidebar.
4. Double-click the system tray icon to show or hide the sidebar instantly. Right-click the icon to open Settings.

## 简体中文

**雷神文件管理 (LS File Manager)** 是一款轻量、灵敏且智能的 Windows 桌面辅助工具，旨在优化日常文件夹和文件的管理效率。当鼠标滑向屏幕最右侧（或左侧）边缘时，侧边栏会流畅地无缝滑出，让您瞬间访问最常用的工作目录、高频图纸和最近查看的项目，而无需在资源管理器中反复寻找。

### 核心特性
- **🚀 边缘悬停唤出**：自动检测鼠标所在的显示器，并在鼠标触碰屏幕边缘时即时滑出；鼠标移开后自动折叠收起。
- **💾 标签状态记忆**：自动记录您最后一次停留的标签页（常用目录、常用文件、最近查看、自定义），并在下次启动时精准恢复。
- **📂 目录文件分流**：对文件夹 and 文件进行独立的修改时间与打开频率排序统计，保证极速呈现。
- **📌 个人自定义栏**：支持手动添加最常用的图纸、设计文件或特定子文件夹，支持一键置顶钉住，完美适配 CAD 等设计工作流。
- **🌓 三种主题色彩**：支持“深色模式”、“浅色模式”及“随系统自动切换”，主题色彩切换无延迟。
- **🌐 全面中英双语**：界面全面支持简体中文与英文，一键无缝切换。
- **⚡ 极简轻量**：基于原生 C# WPF 开发，单文件打包发布仅 60MB 左右，常驻后台几乎不占用 CPU 和系统资源。

### 安装与快速开始
1. 在 GitHub Releases 页面下载最新的发布包 `SidebarApp.exe`。
2. 将程序存放在任意位置，双击即可运行。
3. 将鼠标移至屏幕最右侧（或左侧）边缘，侧边栏会自动滑出。
4. 双击任务栏托盘图标可直接显隐侧边栏，右击托盘图标可以进入“设置”进行自定义宽度、透明度与停靠边缘。
